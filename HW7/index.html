Hero v3.0

  • Originality: Hero v3.0 will have a level-based system (with varying configurations of objects for the bird to stand) and a click-based input, which is unlike any other game that I have built with Phaser.
  • Technical Merit: There is a strong emphasis on gravity and physics. It has to be managed correctly or the level will be difficult to beat.
  • Prototype Postmortem:
    • What did you learn from this prototype? I learned how to use P2 physics, which is a lot more complicated than Arcade physics because game objects are treated as rectangular. Also, I learned how to create levels using multiple states. In addition, I learned a lot about placing game objects using create method's positioning instead of guessing randomly.
    • What was the easiest or hardest part of making it? There was a bug with the left-click triggering event on a Phaser group of child game objects, but I fixed it now. The blocks of yellow can be clicked and destroyed.

  • High Concept: Hero v3.0 is a single-player puzzle platformer.
  • Theme: N/A
  • Mandated Variety:
    • Input: Left-click Mouse
    • Randomness: Deterministic
    • Genre: Puzzle
    • Play Style: Avoid removing obstacles that may cause the bird to fall to its death.
  • Prototype Goal: This prototype is evaluating the enjoyment of click-based pointer games.
  • Player Experience Goals: Players should be able to understand the rules of this game easily and experience a sense of progression after completing levels.
  • Gameplay:Hero v3.0 is a single-input game: left-click. If the object is yellow, then the object will be destroyed if it is clicked. If the object is green, the object cannot be destroyed when it is clicked.
  • Strategies: The objective of this game is to remove all yellow objects without forcing the bird to fall down to its death and touch the ground. Beat level after level until there are no more levels.
  • Story/Setting/Premise: This game is set on mountainous terrain in the middle of the night. The bird finds itself stuck on bricks, and it needs your help removing the obstacles. The player will feel the tension of having to remove obstacles until there is no more without making the bird fall down and die.
  • Target Audience: Ages 5+ & Puzzle Enthusiasts
  • Play Time: 1~2 minutes
