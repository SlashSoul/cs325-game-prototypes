Sergeant Whiskers

  • Originality: Sergeant Whiskers is a Flappy Bird-esque game where the player has three lives and collecting shirts to wash for the laundromat. There is also a way to win the game too.
  • Technical Merit: This game has multiple game states: boot, preloader, main menu, and game. I have also learned how to implement a HP bar.
  • Prototype Postmortem:
    • What did you learn from this prototype? I have gained a better understanding of creating and editing game states in Phaser.
    • What was the easiest or hardest part of making it? This week's storymatic elements were difficult to put together. I also spent some time positioning text and sprites: making small changes and reloading the page is a time killer.
