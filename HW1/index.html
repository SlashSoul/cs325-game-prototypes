Tales of a Purple Alien

  • Originality: There are new items, such as diamonds and first aid kits. There is also an enemy. Platforms were also changed. There is a new stat called 'Health' too.
  • Technical Merit: This prototype has diamonds, first aid kits, and an enemy programmed to kill the player.
  • Prototype Postmortem: I learned how to build a game using JavaScript in Phaser. The hardest part about building this game was learning all the functions in Phaser, especially adding physics to the various sprites and objects in the game.
  • Prototype Assets: Assets were downloaded from here. The background was resized and modified from an image found on Bing Images using Paint. I created some of the assets used in this game, such as the platforms.

  • High Concept: Tales of a Purple Alien is about an alien who has crashlanded on Earth, but he discovers that this planet has rare objects that he wants to bring back with him: stars and diamonds.
  • Theme: The theme is about survival in a strange, unknown world.
  • Mandated Variety: N/A
  • Prototype Goal: The goal of this prototype was to gain an understanding of Phaser and create my first digital browser game using JavaScript.
  • Player Experience Goals: The goal of the player experience is to enjoy this game!
  • Gameplay: The player can use the Arrow Keys on the keyboard in order to move LEFT, RIGHT, and UP.
  • Strategies: The objective of this game is to collect all the stars, diamonds, and first aid kits. Do not touch the bunny.
  • Story/Setting/Premise: Dude is an alien from Phaser who has landed on Earth. He wants to study this planet by collecting the many objects found on Earth. However, there is also a vicious bunny chasing after him.
  • Target Audience: Ages 5+
  • Play Time: < 5 minutes
